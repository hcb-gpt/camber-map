# Generated assets

This folder is generated by the build step.

Outputs:
- `facts.json` — live counts + metadata
- `map.json` — full node/edge graph for the UI
- `map.md` / `map.graphml` — human + graph-tool renderings
- `vp.json` / `vp.prev.json` / `vp.md` — capability health overlay + report
- `changes.json` / `changes.md` — VP diff artifacts

Do not edit these by hand.
